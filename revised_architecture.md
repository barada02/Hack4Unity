# Unity Hub - Revised Architecture Plan

## Core App Structure: 3 Main Sections

### **1. Cultural Discovery** (Private/Personal)
- **Purpose**: User's personal cultural exploration tool
- **Privacy**: Private to user, not social/shared
- **Focus**: Self-discovery and learning about cultures

### **2. Artifact Showcase** (Community/Social)  
- **Purpose**: Creative community platform
- **Social Features**: Like a creative social media
- **Focus**: Generate, share, discover, collaborate on artifacts

### **3. Profile** (User Management)
- **Purpose**: User account, preferences, history
- **Features**: Login, settings, personal stats
- **Focus**: Account management and personalization

## Database Models (MongoDB Collections)

### **Users Collection**
```javascript
{
  _id: ObjectId,
  username: String,
  email: String,
  password_hash: String,
  avatar_url: String,
  created_at: Date,
  profile: {
    location: String,
    bio: String,
    interests: [String]
  },
  stats: {
    artifacts_created: Number,
    cultural_discoveries: Number,
    followers: Number,
    following: Number
  }
}
```

### **Artifacts Collection**
```javascript
{
  _id: ObjectId,
  creator_id: ObjectId,
  title: String,
  description: String,
  prompt: String,           // Original user prompt
  wolfram_code: String,     // Generated Wolfram code
  result_url: String,       // Stored artifact file
  tags: [String],
  created_at: Date,
  stats: {
    likes: Number,
    views: Number,
    remixes: Number,
    comments: Number
  },
  visibility: String        // "public", "unlisted", "draft"
}
```

### **Cultural_Discoveries Collection**
```javascript
{
  _id: ObjectId,
  user_id: ObjectId,
  country1: String,
  country2: String,
  results: {
    similarities: Object,   // Wolfram analysis results
    differences: Object,
    unity_guide: String,
    charts: [String]        // URLs to generated charts
  },
  created_at: Date,
  saved: Boolean
}
```

### **Interactions Collection** (Likes, Comments, Follows)
```javascript
{
  _id: ObjectId,
  user_id: ObjectId,
  target_type: String,      // "artifact", "user", "comment"
  target_id: ObjectId,
  action: String,           // "like", "comment", "follow", "remix"
  content: String,          // For comments
  created_at: Date
}
```
