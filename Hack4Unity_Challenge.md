# Hack4Unity Hackathon Challenge

## About the Challenge

With the world being in such a harsh place, with animosity towards each other based on external factors, we often need to remind ourselves that the world is a good place, a better place when humans work together. This hackathon calls upon aspiring software developers to create a better world through creating an app or software that will create more unity between people!

## Requirements

### What to Build
Build something that will create more unity in the world. It's purposefully a vague prompt. Use this hackathon as an excuse to build the project you have always wanted to make and to create a better more unified world between people!

### What to Submit
A video and answering some questions to be eligible for the winning spot!

## Awards

### Wolfram Award Winners
- **$1,660 in cash**
- **4 winners**

**Special Offer:** Everyone who submits a project using Wolfram Language will receive:
- Wolfram|One Subscription (valued at $1,660 USD)

## Judging Criteria

### 1. Creativity
How creative is your application?

### 2. Solution
Does your application solve the problem?

### 3. Technical Ability
What unique skills did you have to utilize to create the application?
